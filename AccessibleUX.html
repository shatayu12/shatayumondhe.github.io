Urban Ladder

  • Category: A11y Case Study
  • Client: The University of Texas at Austin
  • Project date: 01 Dec, 2022

About the Project

This test was conducted to check the WCAG 2.1 AA conformance of the Urban Ladder Website. In this test, I started off with the easy checks where I conducted various tests such as checking for contrast errors and missing alt text errors etc. and created a list of the Accessibility issues. Based on these primary checks, I used Accessibility tools such as WAVE and ANDI and with the help of which, I evaluated 3 web pages of the Urban Ladder Website. Based on the results of the tools as well as the easy checks, I created a Summary of Changes that was required in order to make that website Accessible. Thus, I re-designed the website and fixed all the issues in the pages that I selected for this test as a part of my case study.
